## Term Project - I2I Elevator

### My Part

디스플레이 파이(Display Pi) 구현을 맡았다.

디스플레이 파이는 다음과 같이 세 가지로 이루어져 있다.

- 카메라
- 터치센서
- PIR

### Functions

디스플레이 파이에서의 카메라는 실시간 스트리밍으로 웹 브라우저 상에서 거울모드로 보여주는 카메라이다. 외부 파이(Exterior Pi)에도 카메라가 있는데, 터치센서를 터치할 때마다 거울모드/외부풍경을 교대로 보여준다. PIR센서는 5초마다 엘레베이터 안에 사람이 있는지 없는지를 감지하여 사람이 있으면 LED 불을 켜고, 사람이 없으면 LED 불을 끈다.(그러고보니 5초면, 사람이 있는 경우 켜지는데 5초 걸리면 답답할 것 같다.. 시간간격을 좀 더 줄이는 것을 고려해봐야겠다.)

(코드에 대한 설명은 추후 추가 예정)

### Issue Track

1. 파이 자체의 성능으로 인해 웹 브라우저를 띄우는 속도가 너무 느리다. => 내 파이가 아닌 다른 사람의 파이로 교체해서 해보는 것을 고려하고 있다.
2. 터치 센서가 전기적인 자극으로 인해 진짜 터치를 한 것이 아님에도 불구하고 터치했다고 우루루 뜰 때가 있다. => 이것도 PIR처럼 타이머를 둬서 일정 시간 간격에 한번만 터치한 것으로 인정한다던가 해야할 것 같다.
